## 🕹️ Jak hrát
 
1. Spusť hru a vyber režim (1 = proti AI, 2 = dva hráči)
2. Deska má políčka číslovaná 1–9:
```
 1 │ 2 │ 3
───┼───┼───
 4 │ 5 │ 6
───┼───┼───
 7 │ 8 │ 9
```
 
3. Zadej číslo políčka kde chceš hrát
4. Vyhráváš když máš 3 znaky v řadě (řádek, sloupec nebo diagonála)
## 🤖 Jak funguje AI
 
Počítač používá algoritmus **Minimax** — projde všechny možné tahy do konce hry a vybere ten nejlepší. AI hraje vždy optimálně, takže ji lze maximálně remizovat.
